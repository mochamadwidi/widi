Tentang Kami

Hi Saya Widi

Setelah menyelesaikan sekolah, Saya bekerja di salah satu perusahaan BUMN. Saya mencari nafkah yang hebat, membeli rumah pertama kami, dan segalanya tampak hebat. Namun, hidup saya menjadi semua tentang lembar penyebaran excel menulis laporan. Saya masih ingat hari ketika saya sedang mengerjakan spreadsheet excel lainnya, dan saya berkata pada diri sendiri, "Saya tidak melakukan ini selama 30 tahun ke depan dalam hidup saya." Saya tahu sesuatu perlu diubah.

Saya memutuskan untuk membeli kamera DSLR pertama saya hanya untuk mencobanya! Saya memulainya dengan mengambil foto putri saya, dan menjadi TEROBSESI dengan fotografi. Saya menyukai segala sesuatu tentang itu. Kemampuan untuk menciptakan seni dan terhubung dengan subjek saya sangat menarik bagi saya. Itu memberi saya outlet kreatif yang saya butuhkan.

Satu dekade dan 300+ pernikahan kemudian, sekarang seperti yang mereka katakan, sisanya adalah sejarah. Saya menganggapnya suatu kehormatan untuk dipercayakan dengan tugas mendokumentasikan beberapa hari terpenting dalam kehidupan klien saya, dan saya tidak sabar untuk mendokumentasikan milik Anda!

FILOSOFI

Saya percaya ketika klien saya mempekerjakan saya, mereka mempekerjakan saya karena kemampuan saya untuk menceritakan kisah mereka melalui lensa saya. Menjadi fotografer pernikahan seni rupa itu berarti saya meluangkan waktu untuk membuat alur cerita untuk klien saya yang akan benar-benar ajaib. Saya ingin klien saya melihat kembali salah satu hari terbesar dalam hidup mereka dan menghidupkan kembali setiap saat. Setiap pasangan berhak mendapatkan pernikahan dongeng mereka. Saya akan memastikan pernikahan Anda didokumentasikan dengan cara yang paling indah. Jika itu mengharuskan saya untuk memasukkan diri saya sendiri untuk membuat momen yang tulus terjadi, itulah yang akan saya lakukan untuk klien saya. Saya seorang fotografer, sutradara, kurator, dan yang terpenting, seorang story teller untuk pernikahan Anda.

PENDEKATAN

Setiap pernikahan itu unik, dan menawarkan kesempatan bagi kisah cinta untuk diceritakan dengan cara yang sama sekali berbeda. Tugas saya adalah mendokumentasikan momen-momen itu sekilas di antara momen-momen, merangkum kenangan dalam bentuk nyata, sehingga klien saya dapat mewariskan ingatan mereka untuk generasi mendatang. Sepanjang sesi potret yang khas, saya akan memimpin klien saya dalam skenario yang berbeda dan membuat mereka bereaksi terhadap saya secara alami. Reaksi selalu menghasilkan emosi yang paling tulus, daripada tindakan. Sebagian besar klien saya memberi tahu saya bahwa mereka bahkan tidak merasa itu adalah sesi potret. Rasanya lebih seperti jalan-jalan Sabtu sore yang khas di taman dengan orang favorit mereka.
